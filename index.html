Bereit für die Auswertung

Lade links die Besuchsliste vom Webserver. Danach siehst du sofort, welche Seiten wie oft aufgerufen wurden und ungefähr, wie viele Besucher du hattest.

Besuche laut Webserver -

Grobe Schätzung: Aufrufe per IP + Browser im 30-Min-Fenster zusammengefasst. Mehrere Personen hinter einer IP zählen als eine.

Seitenaufrufe gesamt -

Echte Seiten — ohne Bilder, CSS oder JS. Vergleichbar mit „Seitenaufrufen" in Google Analytics.

Käufe laut Webserver -

So oft wurde die Danke-Seite gefunden.

So viel sieht Google Analytics -

Von 100 Aufrufen, die dein Webserver zählt, meldet Google Analytics nur so viele.

Seitenaufrufe (wie oft eine Seite geöffnet wurde) zählt der Server genau, soweit sie bei ihm ankommen; Besucher (verschiedene Menschen) sind eine Schätzung — eine Person kann eine Seite mehrfach öffnen, mehrere Leute aus einer Firma zählen als einer. Der Vergleich zeigt Abweichungen in beide Richtungen: Google Analytics weniger → meist Tracking-Verlust (Cookies/Ad-Blocker); der Server weniger → oft Caching/CDN (gecachte Aufrufe erreichen das Server-Log nicht). Käufe sind kaum betroffen, weil die Danke-Seite selten gecacht wird.

Nach der Auswertung steht hier, was du als Nächstes tun solltest.
