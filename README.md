# ServerStory

Sieh in wenigen Minuten, was deine Website laut deinen Server-Logs **wirklich** an
Besuchern und Seitenaufrufen hatte — direkt aus der Quelle, ohne Cookies und ohne
Tracking-Tool. Optional kannst du die Zahlen direkt mit Google Analytics vergleichen
und siehst sofort, ob Google Analytics weniger zählt als die Realität.

Alles läuft lokal in deinem Browser. Deine Datei wird nirgendwohin hochgeladen.

## So benutzt du es

1. Dieses Paket herunterladen und entpacken.
2. **START_HIER.html** doppelklicken — öffnet im Browser.
3. Deine Logdatei (die Besuchsliste vom Webserver, meist `.log` oder `.txt`)
   auswählen und auf **Jetzt auswerten** klicken.

Du bekommst sofort deine meistbesuchten Seiten und wie viele Besucher du im
Zeitraum wirklich hattest.

**Keine Logdatei?** Im Tool gibt es den Button *„Noch keine Datei? Kopier den
fertigen Text hier."* — diesen Text schickst du an deine IT und bekommst die
passende Datei. Zum Ausprobieren gibt es einen **Demo**-Button mit Beispieldaten.

## Datenschutz

Die Auswertung passiert ausschließlich in deinem Browser. Es wird nichts
hochgeladen und nichts an Server gesendet. Angezeigt werden nur Summen — keine
IP-Adressen und keine Nutzerlisten.
